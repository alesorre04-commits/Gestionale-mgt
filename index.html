⚙️ Configura Firebase per la sync
Segui questi passi per abilitare la sincronizzazione in tempo reale:
1. Vai su console.firebase.google.com
2. Crea progetto → Firestore Database → Modalità test
3. Impostazioni → App web → Aggiungi app → copia il config
4. Apri il file HTML, cerca FIREBASE_CONFIG e incolla i valori
5. Pubblica su netlify.com/drop e condividi il link con i colleghi
💡 L'app funziona comunque in locale. Firebase serve solo per la sync tra dispositivi diversi.
